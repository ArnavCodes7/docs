## Subject Management

### Creating Subjects

Create subjects for each course or exam topic:

1. Click **+ Add Subject**
2. Enter subject name (e.g., "Physics")
3. Choose an icon (atom, flask, calculator, etc.)
4. Pick a color gradient
5. Save!

### Subject Properties

| Property | Description |
|----------|-------------|
| Name | Subject title |
| Icon | Visual identifier |
| Gradient | Color theme |
| Exam Template | Linked exam syllabus |
| Study Time | Total time spent |

### Managing Multiple Subjects

- **Tab Navigation** - Switch between subjects
- **Add/Remove** - As needed
- **Reorder** - Drag to change order

---

## Chapter Organization

### Adding Chapters

Within each subject, add chapters:

1. Click **+ Add Chapter**
2. Enter chapter title
3. Chapter appears in the list!

### Chapter Properties

- **Title** - Chapter name
- **Topics** - List of topics within
- **Priority** - High/Medium/Low
- **Study Time** - Time spent
- **Completion** - Percentage complete
- **Revision Status** - Needs revision / Completed

### Chapter Actions

- **Mark for Revision** - Flag for review
- **Add to Tasks** - Create tasks for all topics
- **Edit Title** - Rename chapter
- **Delete** - Remove chapter
- **Reorder** - Drag up/down

## Spaced Repetition System (SRS)

Isotope uses proven **spaced repetition** for optimal retention:

### How SRS Works

1. **Learn a topic** - Study and mark complete
2. **First review** - 1 day later
3. **Second review** - 3 days later
4. **Third review** - 7 days later
5. **Fourth review** - 14 days later
6. **Fifth review** - 30 days later

### Rating System

After each review session, rate your recall:

| Rating | Meaning | Next Review |
|--------|---------|-------------|
| Again | Forgot completely | 1 minute |
| Hard | Struggled to recall | 1 day |
| Good | Recalled with effort | 3 days |
| Easy | Instant recall | 7 days |

### SRS Benefits

- üéØ Optimal retention
- ‚è∞ Less time wasted
- üß† Long-term memory
- üìä Progress tracking

### Where to Use SRS

- In Study page - Review due topics
- In Focus page - Select topics to review
- Notifications remind you when due

2. **Set priorities** - Mark important chapters
