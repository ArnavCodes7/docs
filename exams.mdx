## Exam Timeline

A chronological view of all your exams:

### Timeline View

Shows exams in order:
- Past exams (completed)
- Upcoming exams
- By date

### Exam Card Information

Each exam shows:
- **Title** - Exam name
- **Date** - Scheduled date
- **Days left** - Countdown
- **Priority** - High/Medium/Low
- **Status** - Upcoming/Completed
- **Results** - If completed

### Exam Properties

| Property | Description |
|----------|-------------|
| Title | Exam name |
| Date | Exam date |
| Priority | High/Medium/Low |
| Target Score | Your goal score |
| Syllabus Covered | Topics to be tested |
| Notes | Additional info |

## D-Day Hero Section

The prominent hero section shows your **most important exam**:

### What's Displayed

- **Countdown timer** - Days, hours, minutes
- **Exam name** - Primary exam title
- **Quick stats** - Mocks taken, score trend
- **Actions** - Edit, add result

### Setting D-Day

Any exam can be your "D-Day" (primary exam):
1. Create or edit an exam
2. Toggle "Set as D-Day"
3. It appears prominently here!

## Mock Tests

1. Click **+ Add Mock Test**
2. Enter test details after completing
3. Or link to actual exam

### Mock Test Analytics

- **Score trend** - Improving or declining
- **Subject-wise breakdown** - Strengths/weaknesses
- **Time analysis** - Speed vs accuracy

## Adding & Managing Exams

### Creating an Exam

1. Click **+ Add Exam**
2. Fill in details:
   - Exam name (e.g., "JEE Main 2025")
   - Date and time
   - Priority level
   - Target score
   - Syllabus covered
3. Save

### Editing an Exam

- Click any exam to view details
- Edit fields as needed
- Update status (upcoming → completed)

### Managing Results

After exam:
1. Open the exam
2. Add results:
   - Scored marks
   - Total marks
   - Rank (if available)
   - Percentile
3. Save

### Duplicating Exams

Need a similar exam?
- Click **Duplicate**
- Creates copy with new date

## Tips & Tricks

### Exam Preparation Workflow

1. **Add all exams** - Even small tests
2. **Set D-Day** - Your main focus
3. **Take regular mocks** - Weekly minimum
4. **Analyze results** - Every test
5. **Track trends** - Use analytics
6. **Adjust strategy** - Based on insights

### Managing Multiple Exams

- Prioritize by date
- Set priorities (high/medium/low)
- Use exam countdown
- Link mock tests to exams

### Using Analytics Effectively

- Check weekly trends
- Compare subjects
- Note improvements
- Address weaknesses
