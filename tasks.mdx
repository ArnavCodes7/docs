## Creating Tasks

### Method 1: Floating Action Button

1. Click the **+** button (bottom right)
2. Fill in task details
3. Click **Save**

### Method 2: Quick Add

Use the search bar at the top:

```
"Study Math !p1 #Physics @tomorrow"
```

**Syntax:**
- `!p1` - Priority (p1-p4)
- `#Subject` - Subject tag
- `@when` - Due date

### Method 3: From Study Page

When viewing a topic in Study:
- Click **"Add to Tasks"**
- Task is pre-filled with topic details

## Task Properties

Each task has detailed properties:

### Basic Info
- **Title** - Task name (required)
- **Description** - Detailed notes
- **Subject** - Associated subject
- **Chapter/Topic** - Specific location

### Scheduling
- **Due Date** - When it's due
- **Start Date** - When to start
- **Estimated Time** - How long it takes
- **Energy Level** - High/Medium/Low energy needed

### Priority
- **P1 (Critical)** - Must do
- **P2 (High)** - Should do
- **P3 (Medium)** - Important
- **P4 (Low)** - Nice to do

### Status
- **Backlog** - Not started
- **Todo** - To do
- **In Progress** - Being worked on
- **Done** - Completed
- **Cancelled** - Won't do

### Additional
- **Subtasks** - Break down into steps
- **Recurring** - Daily/Weekly/Monthly repeat
- **Links** - Connect to topics/exams

## Quick Add

The fastest way to create tasks:

### Syntax Guide

| Format | Example | Result |
|--------|---------|--------|
| Basic | `Finish math homework` | Task created |
| Priority | `!p1 Finish math` | P1 priority |
| Subject | `#Math Finish homework` | Tagged Math |
| Due Date | `@tomorrow` | Due tomorrow |
| Due Date | `@next monday` | Due Monday |
| Due Date | `@15/02` | Due Feb 15 |

### Combined Example
```
Study Chapter 5 !p1 #Physics @friday
```
Creates a P1 Physics task due Friday!

## Tips & Tricks

### Task Management Best Practices

1. **Daily Review** - Check tasks every morning
2. **Two-minute rule** - Do small tasks immediately
3. **Batch similar tasks** - Group by subject
4. **Use priorities** - P1 tasks first
5. **Break big tasks** - Use subtasks

### Quick Add Mastery

Master the quick add syntax:

```
# Study !p1 @tomorrow = P1 study task due tomorrow
!p2 #Math @monday = Math task, medium priority, due Monday
```

### Building Habits

- Start with **one** habit
- Keep it small and achievable
- Track every day
- Celebrate streaks

### Productivity Flow

1. **Morning** - Check AI Task Planner
2. **Start with P1** - Most important first
3. **Use Zen Mode** - For deep work
4. **Batch similar** - Group by subject
5. **End with review** - Plan tomorrow
