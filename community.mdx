## Group Discovery

Find study groups that match your interests:

### Browse Groups

- **By subject** - Physics, Math, Chemistry, etc.
- **By exam** - JEE, NEET, CBSE
- **By type** - Competitive, casual, exam-focused
- **By size** - Small groups or large communities

### Group Cards

Each group shows:
- **Name** - Group title
- **Members** - How many members
- **Focus** - Subject/exam focus
- **Activity** - Recent activity level
- **Join button** - Join or request

### Searching

- Search by name
- Filter by criteria
- Sort by popularity/activity

## Study Groups

Collaborate with peers:

### Group Features

| Feature | Description |
|---------|-------------|
| Discussion Board | Chat with members |
| Group Focus | Study together virtually |
| Shared Resources | Notes, links, materials |
| Progress Tracking | See member progress |
| Admin Controls | Manage members/content |

### Joining Groups

1. Find a group you like
2. Click Join
3. Wait for approval (if required)
4. Start participating!

### Creating Groups

Premium users can create groups:
1. Click Create Group
2. Set details:
   - Name
   - Description
   - Focus area
   - Privacy (public/private)
3. Invite members

## Tips & Tricks

### Getting the Most from Community

1. **Join groups** - Find your tribe
2. **Participate** - Engage in discussions
3. **Compete** - Join challenges
4. **Share** - Contribute resources
5. **Help others** - Build reputation

### Building Your Profile

- Complete your profile
- Earn achievements
- Maintain activity
- Help others

### Finding the Right Groups

- Start with subject groups
- Join exam-focused groups
- Try small groups first
- Engage actively
